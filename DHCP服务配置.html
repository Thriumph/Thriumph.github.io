DHCP服务配置

实验目的

(1) 掌握linux系统中主DHCP服务器配置。

(2)掌握dhclient程序的使用方法

项目背景

构建一台DHCP服务器，为局域网中的计算机提供网络参数配置服务。

实验内容

Linux系统下DHCP服务器的配置方法。DHCP客户端工具dhclient的使用方法

步骤

什么是DHCP

DHCP（Dynamic Host Configuration Protocol，动态主机配置协议）通常被应用在大型的局域网络环境中，主要作用是集中的管理、分配IP地址，使网络环境中的主机动态的获得IP地址、Gateway地址、DNS服务器地址等信息，并能够提升地址的使用率

安装DHCP服务软件

相关命令：sudo apt-get install isc-dhcp-server

配置DHCP服务

DHCP的主要配置文件有两个，分别位于 /etc/default/isc-dhcp-server 和 /etc/dhcp/dhcpd.conf。下面要做的就是对这两个文件进行配置

查看本机网络信息

命令：ifconfig

假设ens160网卡为外网网卡
假设ens192网卡为内网网卡

配置isc-dhcp-server

命令：vim /etc/default/isc-dhcp-server

将监听端口INTERFACES修改为内网网卡名称ens192

保存退出

配置hdcpd.conf

相关命令：vim /etc/dhcp/dhcpd.conf

修改配置

启动服务

1.启动DHCP服务

命令: sudo service isc-dhcp-server restart

2.查看DHCP服务是否正常启动

命令：sudo netstat -uap

查看服务列表里是否有dhcpd服务

验证服务

在win10客户端重启网卡，获取IP地址
