Samba服务
更新于 2020-05-11 linux 热度 1 喜欢 0

目的

掌握使用samba进行文件共享服务，局域网下使用samba服务在Linux系统与Windows系统直接共享文件是一项很方便的操作

配置文件

/etc/samba/smb.conf

/usr/bin/smbpasswd

/etc/samba/smbusers

/etc/samba/lmhost

/var/lib/samba/private/passdb.tdb

步骤

安装samba服务器

sudo apt-get install samba samba-common

创建一个用于分享的samba目录

sudo mkdir /home/yangshuming/share

创建的这个目录设置权限

sudo chmod 777 /home/yangshuming/share

添加用户

sudo nano /etc/samba/smb.conf
